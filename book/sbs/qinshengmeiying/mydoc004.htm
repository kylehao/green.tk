第四章　财大气粗够拉风

　

　　在众人企盼中传出喜讯。海豹为之大乐。
　　他昔年成亲较迟，加上先有三位女儿才生下海涛这个宝贝儿子，所以，他比别人渴盼能够早日抱孙。如今，他已达心愿。
　　他便严禁爱子碰岳婷。他更指派一名大夫驻堡为岳婷进补，他更挑选二婢专门侍候岳婷。
　　他当然已派人入京报喜。所以！岳峰夫妇迅速的赶到海家堡。他们见过爱女，便与海豹夫妇欢叙着。岳峰亦趁机提及准备售粮之事，海豹便趁机指点着。
　　三日后，岳峰夫妇便申谢离去。不出一个月，京城三大粮商真携礼一起拜访岳峰，岳峰心中有数，便先与他们打哈哈一翻。然后，他们才要入主题，经过一番讨价还价之后，他们终于谈妥价格。
　　三大粮商按规矩各预付五十万两定金。
　　他们一离去，岳峰亢喜的入房。他不由搂着爱妻。二人大喜着。
　　不久，他二个便赴魏府会见魏全夫妇。四人一叙，便欣然聊着。
　　良久之后，岳峰夫妇方始离去。他们途经银庄，便入内存妥一百五十万两黄金，这是岳峰迄今所获之最高金额，他亢喜之至。
　　他们返府不久，他便亢喜的搂吻爱妻。岳氏魏敏不由边吻边卸衣。不久，二人已上榻快活着。魏敏之母乃是魏全之第三房细姨，她原是京城名妓，所以，未嫁给岳峰便练会一身的床技。所以，她在岳峰面前有发言权。
　　不过，她知道老公之个性，她一直尊重着老公。岳峰因而服服帖帖。两人又快活良久，方始尽兴歇息。这年秋收之后，大批米、麦及高梁一送入京城，三大粮商便直接付钱以及送粮入仓售给中间商。
　　由于安徽上回之水灾，造成粮价上涨，此次之粮价净涨一成。此外，再加上岳峰转手之百分之五利润，京城粮价因而上涨一成半至两成。
　　不出三天，岳峰已售光粮。他只露个脸，便净赚百分之五。由于数量庞大，他因而获利可观。外加那一百五十万两押金每年所产生之九万两利钱，他的身价以及地位因而高涨，他为之大乐。
　　凌鹤不由又妒又气。他的岳父为颜面，便出售粮行。魏全趁机吞并他的粮行。
　　这天下午，岳峰夫妇亲送银票入海家堡。他们欣然申谢着。令他们惊喜的是，海豹把上涨之一成粮与他共享，他因而又获得百分之五利润，他们不由连连申谢。
　　魏敏因而吩咐爱女多孝顺公婆及等候老公。
　　三日后，他们方始离去。他们一返京城，便先入银庄存钱。' 人若走运，城墙也挡不住' ，又过十天，兵中侍良池浩居然自动找媒人上岳府提亲哩！
　　媒人更展示池浩之女池莲之画像。岳峰夫妇不由暗喜。于是，岳峰答允先会面。
　　第三天黄昏时分，池浩夫妇果真率女池莲入岳府拜访，岳峰夫妇便率子迎他们入厅。
　　双方入座不久，便先行品茗。
　　然后，双方客套一番。接着，池浩正色道：' 朝廷正值用人之际，此亲事若成，吾可保举令郎入兵部任吏，以令朗文武皆全，仕途如锦矣！' 岳峰夫妇不由动心。
　　人心便是如此贪，有富便想攀贵，攀贵更不如自己做官，所以，岳峰夫妇及岳胜不由动心。何况，池莲虽不是大美女，也眉清目秀。于是，双方说定这门亲事。池浩便留下爱女之生辰八字及取走岳胜之生辰八字。岳峰夫妇便连夜入魏府报讯。
　　魏全提醒道：' 及早通知海家堡吧！' ' 好！' 于是，岳峰二人欣然离去。
　　翌日，他便择妥吉日送入宫中。当天上午，池浩已派人前天表示同意。于是，岳峰亲自赶向海家堡报讯。
　　海豹一获讯，便先申贺再表示不克参加文定之宴。不过，他表示会参加大喜之宴。岳峰便欣然赶返宫中。
　　十月底，岳峰夫妇便率子携大批礼品入宫与池莲完成文定以及下聘之礼，在场面礼者另有李尚书夫妇。礼成之后，李尚书便与岳峰聊着。池浩当然在旁美言。
　　午前时分，众人便享用酒菜。膳后，岳峰三人便含笑离宫。
　　翌立黄昏，池告便亲入岳府报喜。岳峰上路的先送池浩一份礼，接着，他请池浩送一份礼给李尚书。翌日上午，岳胜便入宫报到。池浩便安排他在兵部由一吏指点着。岳胜人仕之讯因而传遍京城。
　　岳胜将娶兵部侍郎之女，也跟着传出。凌鹤更郁卒啦！人心便如此现实，岳府因而天天高朋满座，他的十八家酒楼生意也成为城民聚集以及捧场之重心。他便经常抽空到各店面巡视。
　　此举既可与熟人招呼提升生意，更可在大庭广大前受众人之巴结，他当然乐此不疲的巡视各店面。他终于尝到露脸的滋味。
　　此外，他吩咐爱子岳胜勤习吏事之外，必须多探听及参加宫中之各种婚丧喜庆活动和多多送礼。因为，他要以黄金帮爱子广结善缘。因为，他已由海家堡学习到这一套公关手法。俗语说：' 人二脚，钱四脚' ，人追钱，即使追得死去活来，也多是一场空，结果身心交疲，怨叹之至！如果钱追人，那便是财源滚滚。
　　岳峰便是一例。麻牧更是明显之例。麻牧上回巧遇苗峰及曹坚拼成同归于尽之后，他因而获得傲世钜富，他光凭利钱收入，便足以骇死人。
　　他自从上回目睹海鹰施展剑招之后，他恍悟知命僧留下之招式乃是剑招，所以，他使天天勤练剑招。他的脑海中更一直萦绕海鹰劈削之威。他练得更勤。他不知不觉的激发出彪悍之气。他不知不觉的符合知命僧之主动攻击精神。他为之剑技大进，他体中之功力由于他的勤练剑招而加速激发出来，他每次皆练出一身的汗，他因而每夜到瀑布冲身洗衣。
　　这夜，他由一株松树下方沿途踏着枝叶练剑而上，然后，他再由上沿途练剑而下，立见松子纷落。他虽然以木剑练招，却仍沿途咻刷大响着。他每提一口气，便由下掠上再沿上掠下。他逼迫自己上下六趟之后，才再换一口气。
　　这夜亥中时分，他练得汗出如浆，方始满意收招。于是，他擒桶及衣物离屋。
　　不久，他一掠落谭旁，便剥光全身。他首先把湿衣裤抛入潭中任它们飘流。接着，他站在瀑布下任由瀑布冲身。

　　别人说，若出汗，不宜骤冲冷水，他却越冲越功力活跃，因为，他每次如此冲身之后，便全身活力十足哩！所以，他如今张腿冲身着。不久，他又觉活力十足。
　　他的小兄弟又如昔般昂举。这根茄子一昂举，便雄伟的骇人！
　　此时，正有一人在远处石后猛咽口水。她便是柳妃。她恨不得能在此时好好的吸吮那根茄子一番，因为，她南征北伐，迄今尚未见过如此硕伟之宝贝哩！她怎会在此时出现于此地呢？
　　原来，她陪九玄子快活一个月余之后，九玄子不但送她六万两黄金，另外送她一瓶灵丹，她便直接返庄。哪知，她返庄一问，柳珊珊根本没有返庄。她怔了一下，便思忖柳珊珊之可能去处。她逐一寻找，她每遇到熟人，便探听消息。
　　她昔年窃丹害九玄子到处找她，她这回却到处找柳珊珊，冥冥之中，果真有一股神秘力量在安排阴阳果报。她已在柳珊珊的身上投入太多的心血，她相信柳珊珊在服下九玄丹之后，必可阴功大成，所以，她定要找到柳珊珊。她今天一入济安城，便先探听着，入夜之后，她便决定重回原点找起。
　　所以，她用过膳便先行歇息。她方才入谷，乍见一人由峰上掠下，便隐在石后。
　　所以，她会瞧见这根宝贝，她一向淫荡，这阵子她为柳珊珊却一直没快活过。
　　所以，她如今已经发浪。她不由思忖该如何快活。不久，她便已经有了主意。
　　她悄悄移动身子。不久，她已躲在那个木桶附近之大石后。她更自地面挑妥两粒石子。
　　不久，麻牧畅然腾空掠向潭旁。他不知情的一掠近，柳妃便弹出二石。叭叭二声，他的双脚一麻，不由啊叫坠下。柳妃格格一笑，便起身掠去接住他。她的右手更直接握上茄子道：' 好宝贝！' ' 你是谁？' ' 好弟弟，听过柳妃否？''　啊，是你！' 麻牧不由又急又厌，因为，他昔年陪恩师支游时，曾经看见柳妃与一人在大庭广众之下打情骂俏以及搂搂抱抱。柯漱石更道出柳妃之淫荡。
　　因此，麻牧对柳妃非常反感。
　　柳妃一落地，便制住麻牧之双脚及仰放于地上。她亢然握吮着茄子。她又捏又吮又舔着。不久，茄子已容光焕发。她不由格格一笑，她浪然剥光全身，她一上马，便沉劲坐下。
　　' 喔！够充实，够饱满，够烫，妙呀！' 她便先扭摇着。她畅然浪语着。麻牧不吭气啦。他暗暗冲着穴道。因为，他知道自己越说越会引来淫声秽语呀！他决定先冲开穴道再宰她。不久，她便以双掌抓着好的左右肩井穴快活着。麻牧方才之心血立即白费。因为，他如今已全身麻软无力呀！
　　柳妃却越挺越起劲。又过不久，她掉头一坐，便以双掌按着他的大腿快活着。
　　他趁机再度冲穴。她却前挺后动及左右扭摇不已！她实在畅透啦！不久，她一转身，又扣肩畅玩着。麻牧不由又全身麻软。不久，她贴乳一抱，便套顶不已！
　　' 她弟弟，及时行乐吧！' 他不吭声的趁机冲穴。不久，她又按上他的双胁道：' 好弟弟，别使坏！' 她便又畅玩着。麻牧不由暗恨遇上这位精明之淫妇。
　　不过，他仍不放弃脱身。又过一个多时辰，柳妃已被茄子弄得全身舒畅，她汗出如浆及浪叫连连，她不由浪然发泄不已！舒畅之中，她已良久未制住麻牧之穴道。
　　麻牧又冲不久，便冲开穴道。
　　他立即抓住她的双胁。
　　' 啊，好弟弟……' ' 贱人！' 他立即把她按倒在地及制住她的双胁。他挥戈连冲。
　　' 好……弟弟……够劲！' ' 贱人，叫吧！叫吧！' 她愤怒的冲刺者。他并非在快活。他在泄恨也！因为，他一生守身如玉，上回睡个午觉便被一名' 三八查某' 欺负，他险些摔入黑坑丢掉一条小命。这回，他被淫妇欺负，他当然不甘心，所以，他怒刺不已！
　　柳妃原本已畅，她又被他冲刺盏茶时间之后，她在浪叫之中，只觉彻骨酥酸以及全身连连的颤抖。她立知不妙。
　　' 好……弟弟……行啦！再玩下去，姐姐会受不了！' ' 去死吧！' 他愤然的全力冲刺。' 啊……好弟弟……饶命……' 她已由浪叫转为求饶。她越求饶，他便越火大。他仍然冲刺着。又过良久，柳妃已泄得晕头转向。她呻吟不已！她全身瘫软。他却仍然怒冲着。终于，柳妃脱阴而亡。她终于遭到报应。她的一身阴功便一涌而入麻牧之水腹。麻牧倏觉' 气海穴' 一胀，急忙收兵。
　　他立见她已一命呜呼哀哉！他立觉腹部胀疼。他急忙下马吸气行功。他迅觉体中增加一批功力。他知道此功力来自柳妃，不由又厌又急。事已至此，他只好继续行功。天亮之后，群鸟才吵醒他。
　　她一吁气，便望向她。他又忖不久，便先入潭净身。然后，他拿起大巾拭身再整装。接着，他拿起衫裙欲替她整装。叭一声，一盒已坠下，他急忙顺手接盒。
　　立见一瓶跳出，他急忙接瓶。
　　不久，他立见瓶中是灵丹。他更发现满盒的银票。此外，他尚找出一些金银。
　　他不客气的全部没收。然后，他把衫裙穿上尸体。
　　不久，他已挟尸掠落草中。他一挥掌，便劈出一坑，不久，他已埋妥柳妃。
　　可真巧，此地正是柳妃上回与九玄子快活之处哩！不久，他匆匆洗净衣裤，便提水离去。他一返屋，便先晾妥衣裤。
　　然后，他取瓶服下六丹。不久，他已专心的行功。立觉功力更充盈及精纯。
　　他便全力行功着。三日后，他才被风雨声吵醒，他一收功，立知功力又进入另一境界，不过，他毫无喜悦，因为，柳妃污他呀！
　　不久，他已入厨洗米炊膳。他顺便烤香肠及腊肉。然后，他取来那盒银票清点着。不久，他已知盒内有二百六十万两金票及三百四十万两银票，他惊喜之余，不由更痛恨柳妃。他便郁卒的用膳。
　　膳后，他便默默的在厅中散步。他不由思忖会不会有人来找柳妃。于是，他关妥门窗又服丹行功着。入夜之后，他便又在松树上下掠纵练剑。他发现功力更精纯及如意运用。他发现招式已更流畅。
　　他又练剑到深夜，方始拎桶及方物离去。
　　他一到潭边旁，便先注意四周。然后，他匆匆沐浴及洗衣。不久，他已拎水及衣物离去。他一返屋，便先晾妥衣裤。然后，他入房服丹行功。他便日夜行功及练剑着。
　　十二月十五日中午，岳胜与池莲风光的在岳府拜堂成亲，贺客不但满庄，更包括大批的官吏及仕绅。海豹夫妇及海涛夫妇亦在场。岳婷一见老哥的风光场面，不由大喜。她知道这一切来自海家堡之助。
　　不久，岳峰已邀请海豹夫妇坐上首座。同席之人尚包括李尚书夫妇。接着，岳胜夫妇也出去敬酒。海豹便向李尚书及池侍郎敬酒。不久，三人便边饮边欢叙着。
　　不出盏茶时间。池浩已邀海豹入书房。两人关妥窗，便低声交谈着。不久，二人已欣然返厅入座。池浩更向李尚书含笑点头。三个便畅饮着。因为，池浩方才已与海豹谈妥合作事宜，自明年起，长城银川及归绥二个边关所需之煤炭、军服及军粮皆由海家供应。
　　海豹已允每年赠十万两白银。他方才更已预付十万两白银。这便是池浩与岳峰结亲之主因。从此，海家堡又增加一条财源。因为，官炭、官粮及官物之价格皆较高呀！
　　海豹喜获此财源，便与二吏畅饮着。这场盛宴便热热闹闹的结束。岳胜与新娘子一返洞房，他虽已微醉，却不敢放肆，池莲更是羞赧的入内室宽衣钻入锦被中。
　　不久，他便轻吻柔抚着。他一直把她哄到春潮泛滥，方始上马。他更轻柔的策马入林。他如履薄冰般侍候她。她终于尝到甜头。两人便合谱青春交响曲。年轻力壮又谙武的他终于把她侍候和欲仙欲死。
　　他一涌入甘泉，他便颤呼道：' 相……公……' 两人便缠绵着。翌日上午，他们便入侍郎府享用归宁宴。海豹夫妇则与岳峰夫妇及海涛夫妇畅饮欢叙着。翌日上午，海豹夫妇四人才欣然离去。海豹一返堡，便召来海鹰及心腹吩咐着。
　　海鹰笑道：' 无心插柳柳成荫也！' ' 哈哈！是呀！料不到会有此财源哩！
　　' 不久，他们的心腹已出去忙碌着。当天晚上，海涛便向爱妻道出此项喜讯。
　　她不由送上香吻。他的火气立旺。不久，她趴在胯间品箫着。他受用的低唔着。
　　又过不久，他更以双峰夹着小兄弟厮磨着。他畅然叫好着。岳婷此次返京获慈母传授此招，她一见老公如此高兴，她便来回的厮磨以及品箫不已啦！终于，他畅然溢出生命之泉。好一口口的吞入腹中。
　　' 好夫人！好夫人！' 他乐透啦！从此，他多了一项宣泄之游戏。他因而爱她入骨。欢娱之中，岳婷顺利的生下一位儿子。
　　海豹乐透啦！他便派人入京报喜。岳婷似皇后般受人侍奉着。海家堡为之贺客如潮。岳峰获信后，不由大乐。因为，此外孙可提升他与海家堡之关系。
　　何况，池莲早已有喜多日呀！他便入宫向池浩报喜。池浩他拖他送赠。当天下午，他便率爱妻启程。这天下午，他们已欣然进入海家堡。不久，海豹欣然率他们入房探视岳婷。不久，岳峰已笑哈哈的抱着婴儿，魏敏便把红包、金链二物系上婴儿之衣上以及小颈上。
　　他们不由赞美着婴儿。海豹为之大乐。不久，岳峰送上池浩之贺礼，海豹便含笑收下。当天晚上，他们便与海鹰夫妇及海涛畅饮欢叙着。他们为喝弥月喜酒，便天天畅游长安胜景。这天中午，海家堡设下一千桌弥月喜宴，近万人欢叙畅饮，海豹父子更逐桌的敬酒招呼着。
　　这一餐，他们醉啦！海鹰也乐透啦！贺客们也醉茫茫的离去啦！岳峰更是又醉又吐着。三日后，他们才启程离去。他们返府之当天晚上，岳胜便表示奉旨将到银川关任将，他更表示此举乃是池侍郎之刻意安排。因为，此举可使他增加历练及便于日后升官。
　　因为，此举可协助海家堡之供应军需品。此外，目前边关平静，正可安稳的做太平官。岳峰夫妇便欣然答允。三日后，岳胜便启程上任，他一上任，周元帅便先安排他管军需品。官官相护，一切皆好办事也。
　　他如昔的猛做公关。他更勤于了解业务。他因而大受欢迎。又过三个月余，他为了了解运补路线以及国界现况，他便在这天上午亲率车队运送粮物离开银川城。
　　白日之炎热及入夜之寒冷，使他见识到关外之艰苦，所幸，官军皆在沿途搭屋及辟泉供人食宿。所以，他在这天下午顺利的抵达飞龙关。
　　石元帅更迎他入帅府品茗欢叙着。因为，任何人皆想巴结他这位朝廷新贵呀！
　　当天晚上，两人更饮酒欢叙着。岳胜的酒量并不佳，他为御寒，加上石元帅频频劝酒，他七饮八喝之下，居然喝得茫酥酥，石元帅便送他入房。
　　他上榻不久，便' 抓兔' 啦！他呕吐一阵子，方始昏沌沌的入眠。官军因今日接到大批粮物，便如昔般在今夜加茶，如今，除巡夜人员之外，众从皆在被窝中酣睡着。
　　子初时分，前方倏地传来惨叫声。竹哨声连呜。钟声急促响着。蹄声如雷。
　　车声隆隆。黝暗的大地突然冲来大批人马及车辆。蒙军展开袭击啦！熄灭十余年之烽火再度燃起啦！
　　蒙军潜练十余年，此次更召集不少的游牧部落共襄盛举，他们更以骑军展开闪电攻击。他们各持火把及利枪疾驰而来。他们凶残的射杀巡夜的官军。他们一冲近军营，便掷出火把及驰向两侧。
　　弓箭手迅疾搭车前一接手。火光一扩大，清晰的照出奔出之官军。万箭齐发，迅即射杀官军。驰向两翼之蒙军迅又折回及掷枪射杀官军。官军为之慌乱后退。
　　不久，蒙军驰于火场各处射杀着。岳胜昏沌的掠出来，正好遇上一批骑军。
　　他抵抗不久，便被五支长枪钉在地上。倏听号角声边鸣，骑军便驰向南方。不久，石元帅已率官军匆匆迎战。
　　弓箭手则趁机入内抢枪箭及食物。官军惊慌迎战半个多时辰，石元帅阵亡。
　　官军为之溃逃。蒙军凶残的追杀着。五万名官军因而大量的伤亡。大批战具及食物便落入蒙军之手。
　　天亮之后，蒙军便大吃大喝的庆贺着。午前时分，二十万大军一到，众人便欢呼着。不久，蒙帅已率大军南下。大批粮具亦由军队跟送南下。幸存的数百名官军便拼命的逃向南方。深夜时分，潜龙关已接获此讯。
　　鲁元帅急忙派人南下求援。官军便连夜备战。二日后，蒙军已排山倒海般出现。
　　鲁元帅见状，便下令固守城池。蒙军便先扎营休息。翌日上午，蒙军正式攻城。官军便固守着。蒙军便以枪箭射攻着。
　　此外，步军更推车送巨木欲撞破城门。官军便拼命射箭掷枪着。急切之中，官军伤亡甚重。黄昏时分，蒙军开始撤退。翌日一早，蒙军便又进攻。两军便又展开' 飞弹攻势'.午后时分，蒙帅派弓箭手密集射箭，步军则推车撞城，官军为护城，因而又造成重大的伤亡。
　　黄错时分，蒙军又无功而退。深夜时分，蒙军便又进攻。官军有备而来，便展开还击。蒙帅见状，便下令退军。翌日，蒙军歇息，蒙帅却召开紧急会议。又过一日，蒙军自一大早便展开密集射箭、掷枪以及来回的推木撞城门，官军也前仆后继的还击着。
　　蒙军便由一大早一直攻到深夜。轰声之中，城门终被撞破。顶在门后之大小物为之纷破。
　　蒙军便冲杀而入。官军便惊慌持截着。双方皆已伤亡累累，官军亏在人数只剩三千余人，蒙军却如潮水般不停的冲杀以及展开包围。不出一个时辰，官军已全部阵亡。
　　蒙军不由先搜索食物及财物。然后，他们好好的补眠一番。翌日下午，驰援的三万名官军在途中遇见报讯的潜龙关军士，他们一听潜龙关已败，便撤向南方。
　　六名骑军更驰向长城各边关报讯及求援。二日后，这批援军已退入天霸关中。
